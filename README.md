# 🗄️ Об'єктно-Орієнтоване Програмування (ООП)

## 📚 Опис
Репозиторій містить лабораторні роботи з курсу **"Об'єктно-орієнтоване програмування"**, ХПК, 3 курс, 1 семестр.
